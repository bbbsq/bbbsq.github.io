Bee浏览器即将发布

2021-7-19

Bee Network Browser (Bee浏览器)是Bee Network DAO核心團隊推出的一款提供去中心 化世界Web体验的应用程序。

除了像现有的Web浏览器一样支持任何Web3.0应用程序外, Bee Network Browser还使人们能够在分散的应用程序与区块链技术集成的应用程序中进行浏览,交互和事务处理,从而获得无缝和友好的用户体验。

Bee Network Browser是一个更加开放的APPS,它会更加的方便、强大,以后会添加 更多的功能,大家可以得到極致的體驗。

在Bee即将突破1400w用户之际,我们将这个诚意之作推荐给各位Bee的先锋,将来它能使任何人在Bee Netwok Browser上访问任何网站,并且我们已经打通了跟主APP的关联,用户只要手机上有Bee主APP,就会拉起授权, Bee Browser上Bee的數量完美的和Bee客戶端的一致,实现互通,接下来很多新开发的功能都会在Bee Browser发布,为了快人一步,大家请关注Bee Browser吧!

現在的第一個测试bata版本,即将在GooglePlay,App Store等應用商店发布下载(预计7月20左右) ,目前已实现了Bee Browser与主APP的数量互通、可以获得Bee、观看新闻等,我们后续会把更多的功能在Bee Browser 更新,比如:錢包、聊天室,超级节点活动等,大家可以敬請期待!

感谢各位先锋对Bee Network的喜爱和支持。
